# Redes-Neuronales

Este proyecto implementa un sistema completo de clasificación automática de dígitos escritos a mano (0–9) utilizando redes neuronales entrenadas sobre el dataset MNIST, además de un conjunto adicional de 50 imágenes generadas manualmente, y finalizando con un sistema de predicción en tiempo real usando cámara web (OCR casero).

El trabajo incluye entrenamiento, evaluación, preprocesamiento de nuevas imágenes, comparación entre conjuntos, mejoras al sistema, y un módulo final para predicción en vivo.
